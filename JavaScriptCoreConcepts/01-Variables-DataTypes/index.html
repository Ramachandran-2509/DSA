üìù Variables Declaration

1. var (Function-scoped, hoisted)

var name = "John"; var age = 25; console.log(name, age);
‚ö†Ô∏è var is function-scoped and can be redeclared. Use let/const in modern JavaScript.

2. let (Block-scoped, can be reassigned)

let city = "New York"; city = "Los Angeles"; // ‚úÖ Can be reassigned console.log(city);

3. const (Block-scoped, cannot be reassigned)

const PI = 3.14159; // PI = 3.14; // ‚ùå Error: Assignment to constant variable console.log(PI);

üî¢ Primitive Data Types

1. Number

let integer = 42; let float = 3.14; let negative = -17; let infinity = Infinity; let notANumber = NaN; console.log(typeof integer); // "number" console.log(typeof float); // "number"

2. String

let single = 'Hello'; let double = "World"; let backticks = `Hello ${single}`; console.log(typeof single); // "string" console.log(backticks); // "Hello Hello"

3. Boolean

let isTrue = true; let isFalse = false; console.log(typeof isTrue); // "boolean"

4. Undefined

let notDefined; console.log(notDefined); // undefined console.log(typeof notDefined); // "undefined"

5. Null

let empty = null; console.log(empty); // null console.log(typeof empty); // "object" (JavaScript quirk!)

6. Symbol (ES6+)

let symbol = Symbol("description"); console.log(typeof symbol); // "symbol"

7. BigInt (ES2020+)

let bigNumber = 9007199254740991n; console.log(typeof bigNumber); // "bigint"

üîÑ Type Conversion

Explicit Conversion

// String to Number let strNum = "42"; let num = Number(strNum); let num2 = parseInt("42.5"); let num3 = parseFloat("42.5"); // Number to String let numStr = 42; let str = String(numStr); let str2 = numStr.toString(); // Boolean Conversion let bool = Boolean(1); // true let bool2 = Boolean(0); // false let bool3 = Boolean(""); // false

Implicit Conversion (Type Coercion)

console.log("5" + 3); // "53" (string concatenation) console.log("5" - 3); // 2 (number subtraction) console.log("5" * "3"); // 15 (number multiplication) console.log(true + 1); // 2 (true becomes 1) console.log(false + 1); // 1 (false becomes 0)

üìù Template Literals (ES6+)

let name = "Alice"; let age = 30; let city = "Paris"; // Old way (string concatenation) let oldWay = "My name is " + name + ", I am " + age + " years old, and I live in " + city + "."; // Template literal way let newWay = `My name is ${name}, I am ${age} years old, and I live in ${city}.`; // Multi-line strings let multiLine = ` This is a multi-line string using template literals. `; // Expressions in template literals let expression = `The sum of 5 and 3 is ${5 + 3}`;

üí° Tips:
  • Use const by default, let when you need to reassign
  • Use descriptive variable names
  • Use UPPERCASE for constants
  • Initialize variables when declaring them
  • Use template literals for string interpolation
