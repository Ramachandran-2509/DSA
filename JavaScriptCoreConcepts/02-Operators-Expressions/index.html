üîß JavaScript Operators & Expressions

‚ûï Arithmetic Operators

Addition (+)

let a = 5 + 3; // 8 let b = "5" + 3; // "53" (string concatenation) let c = 5 + "3"; // "53"

Subtraction (-)

let a = 10 - 3; // 7 let b = "10" - 3; // 7 (type coercion) let c = 10 - "3"; // 7

Multiplication (*)

let a = 4 * 3; // 12 let b = "4" * 3; // 12 let c = 4 * "3"; // 12

Division (/)

let a = 15 / 3; // 5 let b = 10 / 3; // 3.333... let c = 10 / 0; // Infinity

Modulus (%)

let a = 17 % 5; // 2 let b = 10 % 3; // 1 let c = 8 % 2; // 0 (even number)

Exponentiation (**)

let a = 2 ** 3; // 8 let b = 5 ** 2; // 25 let c = 2 ** 0; // 1

üìù Assignment Operators

Basic Assignment (=)

let x = 5; let y = x; // y = 5

Addition Assignment (+=)

let x = 5; x += 3; // x = x + 3 = 8

Subtraction Assignment (-=)

let x = 10; x -= 4; // x = x - 4 = 6

Multiplication Assignment (*=)

let x = 3; x *= 5; // x = x * 5 = 15

Division Assignment (/=)

let x = 20; x /= 4; // x = x / 4 = 5

Modulus Assignment (%=)

let x = 17; x %= 5; // x = x % 5 = 2

üîç Comparison Operators

Equal (==)

5 == "5" // true (type coercion) 5 == 5 // true 5 == 6 // false

Strict Equal (===)

5 === "5" // false (no coercion) 5 === 5 // true null === undefined // false

Not Equal (!=)

5 != "6" // true 5 != 5 // false 5 != "5" // false (coercion)

Strict Not Equal (!==)

5 !== "5" // true (no coercion) 5 !== 6 // true 5 !== 5 // false

Greater Than (>)

10 > 5 // true 5 > 10 // false 5 > 5 // false

Less Than (<)

3 < 7 // true 7 < 3 // false 3 < 3 // false
‚ö†Ô∏è Use strict equality (===) and strict inequality (!==) to avoid unexpected type coercion.

üß† Logical Operators

Logical AND (&&)

true && true // true true && false // false false && true // false false && false // false // Short-circuit evaluation false && console.log("This won't run");

Logical OR (||)

true || true // true true || false // true false || true // true false || false // false // Short-circuit evaluation true || console.log("This won't run");

Logical NOT (!)

!true // false !false // true !0 // true !"" // true !null // true !undefined // true

Nullish Coalescing (??)

null ?? "default" // "default" undefined ?? "default" // "default" 0 ?? "default" // 0 "" ?? "default" // "" false ?? "default" // false

üìà Increment & Decrement Operators

Prefix Increment (++x)

let x = 5; let y = ++x; // x = 6, y = 6 console.log(x, y); // 6, 6

Postfix Increment (x++)

let x = 5; let y = x++; // x = 6, y = 5 console.log(x, y); // 6, 5

Prefix Decrement (--x)

let x = 5; let y = --x; // x = 4, y = 4 console.log(x, y); // 4, 4

Postfix Decrement (x--)

let x = 5; let y = x--; // x = 4, y = 5 console.log(x, y); // 4, 5
‚ö†Ô∏è Be careful with increment/decrement operators. Prefix (++x) increments before use, postfix (x++) increments after use.

üî¢ Bitwise Operators

AND (&)

5 & 3 // 1 // 5 = 101, 3 = 011 // Result: 001 = 1

OR (|)

5 | 3 // 7 // 5 = 101, 3 = 011 // Result: 111 = 7

XOR (^)

5 ^ 3 // 6 // 5 = 101, 3 = 011 // Result: 110 = 6

NOT (~)

~5 // -6 // 5 = 000...101 // ~5 = 111...010 = -6
